## 🧭 Suggested order (if you're starting fresh)

1. **Apex** — the backbone. Almost everything else assumes you understand the data model, SOQL,
   and governor limits.
2. **LWC** — the modern front end. It calls Apex, so do it second.
3. **Data Cloud** — how Salesforce unifies customer data; increasingly the foundation for AI.
4. **Agentforce** — the newest layer. Far easier once you know Apex (custom actions), LWC (custom
   UI), and Data Cloud (grounding/RAG).

If you're prepping for a **specific role**, jump straight to the matching course and finish with
its **Interview Q&A Flashcards** lesson.

---

## 🧠 The whole stack in three sentences

1. **Apex is the brain on the server** — it runs your business logic close to the data, inside
   strict "governor limits" that force efficient, bulk-safe code.
2. **LWC is the face** — lightweight web components that render the UI in the browser and pull
   data declaratively (`@wire`) or imperatively from Apex.
3. **Data Cloud is the memory and Agentforce is the autonomous worker** — Data Cloud unifies every
   customer signal into one profile, and Agentforce reasons over it to take real actions.
